## ‚öôÔ∏è Automation & Validation

### Continuous Integration (CI)

The repository utilizes GitHub Actions to enforce list integrity and deploy the latest versions. The `ci.yml` workflow performs the following critical tasks on every push and daily schedule:

1.  **Syntax Verification:** Ensures all rules conform to AdBlock Plus specifications.
2.  **De-Duplication:** Removes identical entries to optimize list size and parser performance.
3.  **URL Reachability Check:** Verifies that a small subset of critical domains are still resolvable (a canary check against accidental rule errors).
4.  **List Deployment:** If successful, the new lists are committed and available via the raw content URLs.
